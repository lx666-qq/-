面向丁字路口以及十字路口视距盲区安全问题，研发基于STM32F103ZET6与MWR309毫米波雷达的智能预警系统

一、硬件系统组成

本系统由以下核心模块构成，形成多维联动的智能交通预警体系：

1.	主控单元：以STM32F103ZET6微控制器作为核心，负责数据整合与逻辑控制，通过三串口实现雷达、无线模块与显示设备的通信。 

2.	雷达模块：采用MWR309测速雷达，采用毫米波技术实现双向车辆检测（来向与去向），频率24GHz，支持RS485和UART串口(3.3V TTL 电平信号)输出，波特率9600bps (每帧数据包 9 字节)，测速精度达1KM/h，检测距离可达200米以上，测速范围1-240Km/h，速度采用 ASCII 码输出格式。
     
3.	显示单元：采用LED显示屏（P10双色），尺寸1280mm×320mm（由8块32*16的单元板组成），显示“左侧来车”“右侧来车”“注意安全”等动态警示信息。
     
4.	语音模块：使用BY-F800嵌入式工业语音提示器，内置高保真喇叭（≥90dB），支持“左侧来车，注意安全”等定向语音播报，音量可调。
     
5.	无线通讯：选取双AC800IL微功率扩频无线模块，实现设备间数据同步与远程管控，传输距离≥1km。
     
6.	报警模块：爆闪灯、警示灯和喇叭通过四路继电器模块连接供电，方便主控的IO口进行控制，响应时间≤20ms。
     
二、系统功能

1.实时监测与数据处理

（1）通过STM32F103ZET6开发板的串口2（）接收MWR309雷达获取的车辆以及行人的速度、方向数据（检测精度±1km/h），使用串口5接收AC800IL天线模块接收到的另一个雷达发送的速度以及方向信息。

（2）STM32F103ZET6主控芯片实时分析数据流，通过接收到的雷达和天线的数据改变程序标志位，使主控芯片向串口1发送LED显示屏的字模显示数组，从而改变LED屏幕的显示字幕，切换不同场景的警示标语。

2.多功能警示输出

（1）视觉警示：LED屏动态切换显示内容，如“左侧来车”，支持自定义字模。

（2）语音警示：BY-F800根据预警等级播放差异化语音（如“右侧来车”），支持方言播报与音量自适应调节。

（3）警示灯：继电器控制爆闪灯同步闪烁，提升夜间及恶劣天气下的警示效果。

3.多种类型产品

（1）单机部署模式：系统配置单个测速雷达与单个LED显示屏。雷达部署于主干道监测支路（如右转专用道）的行人与车辆动态，实时通过无线通信将检测数据发送至支路交叉口处的LED显示屏。该显示屏同步向主干道车辆展示预警信息（如“右向来车，注意安全”）及实时车速，形成单向信息闭环。如图一所示。
 
（2）双机部署模式：系统升级为双雷达双屏配置。两个测速雷达分别安装于主干道两侧的支路入口处（如左/右转车道），同步检测双向来车及行人数据。采集信息通过5.8GHz定向天线传输至主干道中央的LED双联显示屏，分区域显示两侧支路的实时交通状态（如“左侧来车减速”、“右路行人通过”），实现双向预警联动。如图二所示。
